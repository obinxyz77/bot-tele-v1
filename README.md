# 🌸 OBINXYZ TELEGRAM BOT 🌸

Bot Telegram multi-fitur menggunakan Node.js

## ✨ Fitur
- /start → Pesan sambutan
- /menu → Menampilkan semua fitur
- /cekid → Lihat ID Telegram pengguna
- /info → Informasi bot
- /randomquote → Dapatkan kutipan acak
- /weather <kota> → Cek cuaca
